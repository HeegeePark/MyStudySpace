# 기본 자료구조

## 배열(Array)
- 장점 : i번째 원소를 바로 알 수 있음.(List[i])
- 단점 : 원소의 추가/삭제가 까다로움

## 링크드 리스트(Linked List)
- 장점 : 원소의 추가/삭제가 간단함.
- 단점 : i번째 원소를 알기가 까다로움.

## 캡슐화(Encapsulation) <- **매우 중요**
- 자료구조를 사용하는 사람은 자료구조가 어떻게 동작하는지 알 필요 X
	- 컴퓨터 내부적으로 어떻게 동작하는지, 어떤 자료가 어떻게 저장되는지 등등
- 그래서 블랙박스처럼 내부적 처리를 숨김.
- 이를 "구조체"로 구현

## 구조체(Struct)
- 하나의 Black box를 구현할 때 사용.
- Type을 정의.
- 멤버변수 : 구조체 안에서 선언된 변수
- 메소드 : 구조체 안에서 구현된 함수
