# TableView와 CollectionView의 차이점을 설명하시오.

UIScrollView의 서브클래스로 무한한 양의(unbounded) 정보에 무한한 접근을 제공하기 위해 사용된다. 하지만 자료를 보여주는 방식에서 차이가 발생한다.

- Table View: 정보를 하나의 긴 리스트로 보여준다.(섹션 분리는 가능)
- Collection View: 다양한 방식이 가능하지만 기본적으로 정보를 2D 포맷으로 보여준다. 글을 읽듯이 왼쪽에서 오른쪽으로, 그리고 다음 줄로 이어서 계속 작성하는 방식을 말합니다. 컬렉션 뷰는 커스터마이징이 가능하다.

- 결국 두 View의 차이는 하나는 행으로 나열하여 자료를 보여주는 것이고, 다른 하나는 2차원으로 놓여진 무작위의(arbitrary) 아이템이라는 것
